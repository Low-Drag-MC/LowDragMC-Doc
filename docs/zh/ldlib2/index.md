# 简介
{{ version_badge("2.0.0", label="自", icon="tag") }}

LDLib2 是对原版 LDLib 的完全重写，从头开始重新设计，以支持现代版本的 Minecraft。
它在模组开发的多个领域提供了先进的高级解决方案，显著降低了开发复杂度和长期维护成本。

LDLib2 拥有庞大且结构良好的代码库，为 UI 系统、着色器（shader）、模型渲染、数据同步与持久化以及游戏内视觉编辑器提供了强大的基础设施，使开发者能够更高效地构建复杂系统。

## 💡 与 [LDLib](../ldlib/index.md) 的主要区别
* 完全重新设计的架构
所有核心系统都已重写，以与现代 Minecraft 版本的内部结构保持一致，从而生成更清晰、更可靠的代码。
* 全面的文档
LDLib2 包含详细的文档、注释和示例代码，解决了 LDLib 长期以来实现不清晰、难以阅读的问题。
* 移除遗留系统
许多过时或未使用的框架已被移除，使 LDLib2 更加轻量且易于维护。
* 改进的模组兼容性
为主流模组（如 JEI、KubeJS 和 EMI 等）提供更稳定、更可靠的集成。
