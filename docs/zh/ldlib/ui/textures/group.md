# GuiTextureGroup

## 基本属性

| 字段     | 说明                                       |
|----------|--------------------------------------------|
| textures | 组中使用的纹理数组                         |

---

## API

### setTextures

设置组中使用的纹理。

=== "Java / KubeJS"

    ``` java
    guiTextureGroup.setTextures(texture1, texture2);
    ```

---

### setColor

设置组中所有纹理的颜色。

=== "Java / KubeJS"

    ``` java
    guiTextureGroup.setColor(0xff00ff);
    ```
