# 介绍
{{ version_badge("2.1.0", label="Since", icon="tag") }}

LDLib2 提供了基于 [`Taffy`](https://taffylayout.com/) 布局引擎的强大 UI 库。
如果你在 GUI 开发方面遇到困难，LDLib2 绝对是你的最佳选择之一。与 LDLib 相比，LDLib2 进行了彻底的重构：

* 现代 UI 布局系统
* 现代 UI 事件系统
* 类似 CSS 的样式表系统
* 数据绑定 + RPC 事件
* 大量即插即用组件
* XML 支持 + 游戏内 UI 可视化编辑器
* XEI / KJS 支持
* 完整的文档和使用示例
