# 介绍{{ version_badge("2.1.0", label="Since", icon="tag") }}
LDLib2提供了基于[`Taffy`](https://taffylayout.com/)布局引擎的强大的UI库。如果您在 GUI 开发方面遇到困难，LDLib2 绝对是您的最佳选择之一。与LDLib相比，LDLib2进行了彻底的重构：
* 现代UI布局系统* 现代 UI 事件系统* 类似 CSS 的样式表系统* 数据绑定+rpc事件* 大量即插即用组件* xml 支持 + 游戏内 UI 可视化编辑器* xei / kjs 支持* 完成的文档和使用示例
