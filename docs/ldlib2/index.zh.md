# 介绍{{ version_badge("2.0.0", label="Since", icon="tag") }}
LDLib2 是对原始 LDLib 的完全重写，从头开始重新设计以支持现代版本的 Minecraft。它在模组开发的许多领域提供先进的高级解决方案，显着降低开发复杂性和长期维护成本。
LDLib2 具有一个大型、结构良好的代码库，为 UI 系统、着色器、模型渲染、数据同步和持久性以及游戏内可视化编辑器提供强大的基础设施，使开发人员能够更有效地构建复杂的系统。

## 💡 与[LDLib](../ldlib/index.md) 的主要区别* 完全重新设计的架构所有核心系统都经过重写，以符合现代 Minecraft 版本的内部结构，从而产生更清晰、更可靠的代码。* 全面的文档LDLib2 包含详细的文档、注释和示例代码，解决了 LDLib 长期存在的实现不清楚和难以阅读的问题。* 删除遗留系统许多过时或未使用的框架已被删除，使 LDLib2 更加轻量级且更易于维护。* 改进了模组兼容性为JEI、KubeJS、EMI等主流mod提供更稳定稳定的集成。
